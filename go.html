Using the short variable declaration operator

firstName := "Wei-Meng"

declare and initialize multiple variables (of different types) in a single statement

firstName, lastName, age := "Wei-Meng", "Lee", 25

using var key word

var firstName, lastName string = "Wei-Meng", "Lee"

Note that if you’re declaring and initializing multiple variables using the var key word, all the variables must be of the same type. The following is not allowed:

var firstName, lastName string, age int = "Wei-Meng", "Lee", 25

The preceding statement can be fixed by removing the data type, like this:

var firstName, lastName, age = "Wei-Meng", "Lee", 25

Alternatively, you can declare them this way

var (
        firstName string = "Wei-Meng"
        lastName  string = "Lee"
        age       int    = 25
    )

When you declare and initialize a variable using the := operator, you can’t use it for variables declared outside function bodies, like the following:

package main

import "fmt"
num1 := 5 // error; non-declaration statement outside function body

func main() {
    fmt.Println("Hello, world!")

}

The := operator can only be used for declaring and initializing variables inside functions

Removing Unused Variables

The presence of an unused variable may indicate a bug, while unused imports just slow down compilation. Accumulate enough unused imports in your code tree and things can get very slow. For these reasons, Go allows neither.

if a variable is declared outside a function and not used any- where in the program, the compiler wouldn’t complain:

If you insist on having unused variables in your program, Go offers a quick way to address this issue. Simply assign the unused variable to a blank identifier (_), like this

package main

import "fmt"

func main() {
    var num1 = 5
    _ = num1 // The compiler is now happy!
    fmt.Println(num1)
}

Dealing with Strings

A string can contain special characters (such as \n or \t), as shown in the following example:

package main

import "fmt"

func main() {
    address := "The White House\n1600 Pennsylvania Avenue NW\nWashington, DC 20500"

Go also supports raw strings. A raw string is enclosed by a pair of back ticks (``). It can span multiple lines, and special characters have no meaning in it. Here’s an example of a raw string:

package main

    quotation := `ali
saber
ali
ghttas
`
    fmt.Print(quotation)

}

To convert string values to specific types, such as Boolean, floating point num- bers, or integers, you can use the various Parse functions:

package main

import (
    "fmt"
    "strconv"
)

func main() {
    //str to bool
    b, err := strconv.ParseBool("t")
    fmt.Println(b)        // true
    fmt.Println(err)      // <nil>
    fmt.Printf("%T\n", b) // bool

    //str to float
    f, err := strconv.ParseFloat("3.1415", 64)
    fmt.Println(f)        // 3.1415
    fmt.Println(err)      // <nil>
    fmt.Printf("%T\n", f) // float64

    // str to int64
    i, err := strconv.ParseInt("-18.56", 10, 64) // base
    fmt.Println(i)                               // 0
    fmt.Println(err)                             // strconv.ParseInt: parsing "-18.56": invalid syntax
    fmt.Printf("%T\n", i)                        // int64

    //str to uint
    u1, err := strconv.ParseUint("18", 10, 64)
    fmt.Println(u1)        // 18
    fmt.Println(err)       // <nil>
    fmt.Printf("%T\n", u1) // uint64

    //str to to uint
    u2, err := strconv.ParseUint("-18", 10, 64)
    fmt.Println(u2)        // 0
    fmt.Println(err)       // strconv.ParseUint: parsing "-18": invalid syntax
    fmt.Printf("%T\n", u2) // uint64

}

To convert between the various numeric data types like int and float, you can simply use the int(), float32(), and float64() functions:

package main

import "fmt"

func main() {
    num1 := 5
    num2 := float32(num1)
    num3 := float64(num2)
    num4 := float32(num3)
    num5 := int(num4)
    fmt.Printf("%T\n", num1) // int
    fmt.Printf("%T\n", num2) // float32
    fmt.Printf("%T\n", num3) // float64
    fmt.Printf("%T\n", num4) // float32
    fmt.Printf("%T\n", num5) // int
}

Interpolating strings

The Sprintf() function formats a string based on the formatting verbs (such as %d and %s).

package main

import "fmt"

func main() {
    queue := 5
    name := "John"
    s := fmt.Sprintf("%s, your queue number is %d", name, queue) // Sprintf formats according to a format specifier and returns the resulting string
    fmt.Println(s)

}
